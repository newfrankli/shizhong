# 天气时钟 (Weather Clock)

一个基于 PyQt5 开发的桌面天气时钟应用，具有简洁美观的界面和实用的功能。

## 功能特点

- 🕒 实时显示时间和日期
- 🌤 显示当前天气信息（温度、天气状况）
- 📍 自动获取位置或手动设置城市
- 🎨 支持自定义字体和大小
- 📏 可调整窗口大小和透明度
- 🖱 支持鼠标拖动调整位置
- 🔒 支持锁定位置和大小
- 🚀 支持开机自启动
- 📁 系统托盘图标，方便快速访问

## 技术栈

- Python 3.x
- PyQt5
- requests

## 安装和运行

### 直接运行可执行文件

1. 从 `dist` 目录中找到 `clock.exe` 文件
2. 双击运行即可

### 从源代码运行

1. 确保已安装 Python 3.x
2. 安装依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行主程序：
   ```bash
   python clock.py
   ```

## 配置说明

### 首次使用

1. 右键点击系统托盘图标，选择「设置」
2. 填入和风天气 API Key 和 API Host
3. 可选：设置自定义城市（留空则自动获取位置）
4. 点击「保存」，天气信息将自动更新

### 自定义设置

- **调整大小**：右键拖动窗口可调整大小
- **拖动位置**：左键拖动窗口可改变位置
- **锁定设置**：在托盘菜单中选择「锁定位置/缩放」
- **自定义字体**：在托盘菜单的「自定义字体」中选择要修改的元素
- **调整大小**：在托盘菜单的「调整大小」中修改各元素大小

## 打包说明

使用 PyInstaller 打包应用：

```bash
pyinstaller clock.spec
```

打包后可执行文件将生成在 `dist` 目录中。

## 注意事项

1. 天气信息需要依赖和风天气 API，请确保已获取有效的 API Key
2. 首次运行需要配置 API 信息才能显示天气
3. 如果无法获取天气，将显示缓存的天气数据或离线状态
4. 建议将应用设置为开机自启动，以便持续使用
